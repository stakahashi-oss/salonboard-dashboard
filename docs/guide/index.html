AI理事会とは
7名の専門AIが会議形式で課題を解決する

「理事会起動」と入力するだけで、7名のAI専門家が会議形式でサロン経営の課題を分析し、具体的な施策を提案してくれます。箇条書きではなく、専門家同士が対話しながら答えを出すスタイルです。

💡
施策は必ず 課題→原因→施策→期待効果 の4点セットで提示されます。「なんとなく良さそう」ではなく、根拠のある提案が返ってきます。
🧭
Strategist（戦略家）
事業計画・中長期戦略・競合ポジショニング
🧠
Psychologist（心理学者）
顧客感情分析・スタッフ心理・行動変容
📣
Marketer（マーケター）
HPB・クーポン設計・SNS・集客入口
💎
LTV_Architect（LTV設計家）
回数券・再来設計・顧客生涯価値最大化
🔬
Researcher（調査員）
競合監視・市場データ・口コミ分析
🎨
UX_Designer（UX設計家）
カウンセリング体験・オペレーション改善
⚡
CTO（技術責任者）
システム構成・自動化・LINE API

導入手順
Mac対応 / 所要時間：約10分
⚠️
料金についてClaude Codeの利用には月額 $20（約3,000円）が必要です。アカウントは 個人名義で作成 してください。費用負担については担当者にご確認ください。
1
Anthropicアカウントを作成する
claude.ai にアクセスしてアカウントを作成します。メールアドレスまたはGoogleアカウントで登録できます。
⚠️ このアカウントを作成した方が、AI出力の最終確認責任者になります。
🔗
登録URL：https://claude.ai
2
Claude Code をインストールする
Macの「ターミナル」アプリを開いて、以下のコマンドを貼り付けてEnterキーを押します。
curl -fsSL https://claude.ai/install.sh | bash
3
ログインする
ターミナルで以下を実行するとブラウザが開きます。Step 1で作成したアカウントでログインしてください。
claude
4
設定ファイルを取得する
AI理事会の設定ファイルをダウンロードします。以下をそのまま実行してください。
mkdir -p ~/.claude && curl -o ~/.claude/CLAUDE.md https://raw.githubusercontent.com/stakahashi-oss/salonboard-dashboard/main/CLAUDE_template.md
5
理事会を起動する
ターミナルで claude を起動し、チャット欄に入力します。
claude
チャット画面が開いたら：
理事会起動
3
使い方のコツ
情報を具体的に伝えるほど精度が上がる
✅
良い質問（具体的な数字・状況を伝える） 「〇〇店の今月売上は△△円で目標まで□□万円足りない。残り5日で何ができるか」
「このLINE文章はお客様が来たいと思う内容か確認して」
「再来率が8%しかない。原因と改善策を教えて」
❌
避けた方がいい質問（情報が少なすぎる） 「何かいい施策ありますか？」
「売上を上げるには？」
4
活用例
実際の会話イメージ

例①：LINE配信文章のチェック

あなた
理事会起動。この配信文章はお客様が来たいと思う内容かな？（文章を貼り付ける）
AI理事会
🧠 Psychologist：まず冒頭の「空き情報のお知らせ」というタイトルが問題です。「空きがある＝人気がない」と読まれるリスクがあります…
📣 Marketer：心理学者の指摘に加えて、割引金額を冒頭に出すことで離脱を防げます…

例②：空き枠を埋める施策を考える

あなた
理事会起動。今週土日の予約が42%しか埋まっていない。LINE配信で空き枠を埋めたい。LINE登録者は673人。
AI理事会
🧭 Strategist：残り2日という時間軸を考えると、施策は「今いる顧客を動かす」一択です…
📣 Marketer：配信は今日の21時台が最適です。スマホ確認のピーク時間帯で開封率が最も高くなります…
5
利用ルール
必ず守ってください

⚖️ アカウント作成者が最終確認責任者

このツールはAIが提案を行いますが、最終的な判断・確認・実行の責任はアカウントを作成したご本人にあります。

AIの提案をそのまま使用するのではなく、必ず内容が正しいか・社内ルールに沿っているかを自分で確認した上で実行してください。特に顧客への配信・価格変更・外部への共有を行う際は、必ず確認を取ってから実施してください。

🚫 入力してはいけない情報
  • 顧客の氏名・電話番号・メールアドレスなど個人情報
  • 社外秘と指定されている数値・資料
  • 他社の機密情報
⚠️
AIの出力は参考情報です
AI理事会の提案はあくまで参考です。実際の意思決定は人間が行ってください。数値の予測・効果見込みは保証されるものではありません。
