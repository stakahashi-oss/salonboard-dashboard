3. 全店共通の構造的問題

① PV数の全店低下
仮説: ビビビサイトの影響。ただし数値検証未完了。タカショーが確認すべき。
② 回数券販売率の全店低下（川崎除く）
ビビビサイト経由で回数券クーポンを出していた影響の可能性。
③ メニュー比率の偏り（眉毛ワックス単品30〜40%超）
あきほさん:「メニュー比率をクロスセルで上げるのはめちゃめちゃ無謀。スタッフに言わせるのではなく、クーポン設計・入口の仕組みで変えるべき。セットメニューのクーポンだけ安く出す、不要な単品は制限する等。入口を変えるのがタカショーのポジションでできること」
④ 新規→リピート動線の未整備（根本課題）
天龍:「安いクーポンで入客するまではいい。問題はその後の動線。¥6,600で来た人が次回¥7,400になるのはハードルが高すぎる。回数券で3回目まで差額¥300になる設計をワンセットで落とし込まないと、ただの垂れ流し」

6. タカショーへ：最も意識して改善すべきこと

1.「作業」で終わらせるな。「思考」を出せ

施策が「課題の原因」とセットになっていないことが最大の指摘。

❌ NG: 「客単価が低いです → オプションセット配信します」
✅ OK: 「客単価が低い → 原因は回数券消費時に0円で帰しているから → 回数券利用者向けにオプション限定配信 → 月¥3万→¥8万の改善見込み」

2. 自分のポジションでできることを理解しろ

タカショーができるのは入口（集客・クーポン設計・配信）。現場オペレーション（回数券の売り方、指名獲得、クロスセル）はSVと現場スタッフの仕事。自分の領域外の課題が出てきたら切り分ける。ただし、事前にSVとすり合わせておくのはタカショーの仕事。

3.「仕組み」で変えろ。スタッフに頼るな

  • 眉毛ワックス単品40% → セットクーポンだけ安く出して単品クーポンを制限する
  • 再来率が低い → 2回目限定クーポンを設計して自動的に再来動線を作る

4. 施策の弾は温存しろ

1ヶ月に配信は1〜2本に絞る。今月だけで全部出すと来月何もなくなる。お客様も情報過多で何が何か分からなくなる。
